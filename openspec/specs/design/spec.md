## Requirements
### Requirement: Modern Visual Identity
The application SHALL use a refined "Red Tourism" aesthetic that balances historical gravity with modern web design trends.

#### Scenario: Color Palette
- **WHEN** viewing the application
- **THEN** the primary colors SHALL be Deep Red (#8B0000) and Gold (#FFD700) accents, but balanced with plenty of whitespace/light gray to prevent visual fatigue.

#### Scenario: Typography
- **WHEN** reading text
- **THEN** the font SHALL be a modern sans-serif with good support for Simplified Chinese (e.g., Noto Sans SC).

### Requirement: Fluid Interactions
The interface SHALL respond to user input with smooth transitions.

#### Scenario: List Selection
- **WHEN** clicking a route or place
- **THEN** the active state SHALL animate into view (e.g., background color fade, scale transform).
